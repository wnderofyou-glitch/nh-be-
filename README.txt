NH&BE Mini App — демонстрационный фронтенд-прототип для диплома.

Как открыть:
1. Распакуйте архив.
2. Откройте файл index.html двойным кликом в браузере.
3. Лучше проверять в режиме телефона: F12 → иконка мобильного устройства.

Что уже работает:
- нижнее меню;
- главная страница;
- мини-маркет;
- поиск и категории товаров;
- добавление товаров в корзину;
- изменение количества товаров;
- удаление товаров;
- оформление заказа с демо-сообщением;
- раздел услуг;
- модальные окна «Подробнее»;
- визуальный профиль клиента;
- сохранение корзины в localStorage.

Где менять товары:
- script.js → массив products.

Где менять услуги:
- script.js → массив services.

Где менять цвета:
- style.css → блок :root в самом начале файла.

Где менять картинки:
- папка assets.
- В script.js у каждого товара есть поле image.

Важно:
Это прототип без сервера и базы данных. Для защиты диплома он демонстрирует пользовательский путь: просмотр, выбор, корзина и оформление заказа.
